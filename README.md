# Uber Ride Analytics Dashboard

## Overview
The Uber Ride Analytics Dashboard is an interactive Power BI dashboard designed to analyze Uber ride data and uncover insights related to bookings, revenue, customer behavior, and operational performance.

## Objectives
- Analyze completed and lost bookings
- Track revenue and distance metrics
- Understand demand trends across quarters
- Compare revenue by vehicle type
- Identify top pickup and drop locations
- Evaluate customer and driver ratings

## Tools Used
- Power BI Desktop
- Dataset: Uber ride data (CSV/Excel)
- GitHub

## Key Metrics
- Completed Bookings: 17K
- Lost Bookings: 10K
- Total Revenue: 9M
- Total Distance Covered: 450.79K
- Average Trip Distance: 24.61
- Average Customer Rating: 4.41
- Average Driver Rating: 4.23

## Key Insights
- Highest booking activity observed in Q2
- Demand drops in Q3 and recovers in Q4
- Go Sedan and Go Mini generate major revenue
- Ashram and Rajouri Garden are top pickup locations
- Narsinghpur is the most common drop location

## Dashboard Features
- Interactive slicers (Month / Quarter)
- KPI cards for quick insights
- Time-based trend analysis
- Vehicle-wise revenue breakdown
- Location-based analysis

## How to View the Dashboard
1. Download the `.pbix` file from this repository
2. Open it using Power BI Desktop
